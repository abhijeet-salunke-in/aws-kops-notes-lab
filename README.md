# AWS Study Notes and Configurations

A structured collection of AWS study notes, hands-on labs, service configurations, architecture diagrams, CLI commands, troubleshooting guides, and best practices.

## Purpose

This repository is designed for:

- AWS Certification Preparation
- Cloud Engineering Learning
- DevOps Practice
- Infrastructure Documentation
- Interview Preparation
- Real-World AWS Deployments

| Category | Services |
|-----------|-----------|
| Identity | IAM, Organizations |
| Compute | EC2, Lambda, ECS, EKS |
| Storage | S3, EBS, EFS |
| Database | RDS, DynamoDB |
| Networking | VPC, Route53, CloudFront |
| Monitoring | CloudWatch, CloudTrail |
| Security | KMS, Secrets Manager, WAF |
| IaC | CloudFormation, Terraform |
| DevOps | CI/CD, Automation |
